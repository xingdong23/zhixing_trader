# 知行交易系统 - 全新设计系统 V2.0

## 🎨 设计理念变革

### 从深色科技风到现代简洁风
- **原设计**: 深色主题 + 科技蓝配色 + 霓虹效果
- **新设计**: 浅色主题 + 优雅紫色主题 + 现代简洁

### 核心设计原则
1. **简洁优雅** - 去除多余装饰，专注内容本身
2. **现代时尚** - 采用当前流行的设计趋势
3. **清晰易读** - 确保所有文字清晰可读 (用户特殊需求)
4. **一致体验** - 统一的视觉语言和交互模式

### 组件设计语言

#### 按钮系统
- **主要按钮**: 渐变背景 + 轻微阴影 + 悬浮动效
- **次要按钮**: 边框设计 + 悬浮变色
- **幽灵按钮**: 透明背景 + 悬浮高亮
- **渐变按钮**: 彩色渐变 + 光影效果

#### 卡片系统
- **默认卡片**: 浅色背景 + 精致边框 + 柔和阴影
- **统计卡片**: 渐变背景 + 悬浮动效 + 放大交互
- **功能卡片**: 边框强调 + 光晕效果 + 提升动画
- **玻璃卡片**: 毛玻璃效果 + 半透明背景

#### 表单系统
- **输入框**: 圆角设计 + 聚焦边框 + 底部装饰线
- **选择器**: 现代化下拉箭头 + 渐变边框
- **文本域**: 可调整大小 + 智能高度
- **标签**: 优雅字体 + 必填标识
