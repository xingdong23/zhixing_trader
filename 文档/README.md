## 📊 系统概览

```
Zhixing Trader V2.0
├── Trading Journal (8001)   - 交易日志模块
├── Quant Trading (8002)     - 量化交易模块
└── Market Data Service      - 市场数据服务
```

### 服务端口
| 服务 | 端口 | 功能 |
|------|------|------|
| Trading Journal | 8001 | 交易记录、股票管理、数据同步 |
| Quant Trading | 8002 | 策略开发、策略执行、回测 |
| Frontend | 3000 | 前端界面 |
