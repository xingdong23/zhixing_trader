✅ 多文件数据加载功能已实现

═══════════════════════════════════════════════════════════════

【功能概述】

回测系统现在支持多种数据源格式：
1. 单个文件
2. 多个文件列表
3. 通配符匹配
4. 目录加载

所有数据会自动合并、按时间排序、去重处理。

═══════════════════════════════════════════════════════════════

【配置示例】

1. 单个文件
{
  "data": {
    "source": "data/ETH-USDT-1m-2025.10.23.csv"
  }
}

2. 多个文件
{
  "data": {
    "source": [
      "data/ETH-USDT-1m-2025.10.21.csv",
      "data/ETH-USDT-1m-2025.10.22.csv",
      "data/ETH-USDT-1m-2025.10.23.csv"
    ]
  }
}

3. 通配符
{
  "data": {
    "source": "data/ETH-USDT-*.csv"
  }
}

4. 目录
{
  "data": {
    "source": "data/"
  }
}

═══════════════════════════════════════════════════════════════

【使用方法】

1. 准备数据文件
   放入 backtest/data/ 目录

2. 编辑配置文件
   backtest/configs/backtest_config.json

3. 运行回测
   python backtest/run_backtest.py
