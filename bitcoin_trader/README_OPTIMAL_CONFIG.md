## 📊 交易逻辑

### 入场条件
- 价格站上EMA13
- EMA5 > EMA13 > EMA21（多头排列）
- 使用85%仓位，2.7倍杠杆

### 出场条件
1. **硬止损**: 亏损达到3.2%
2. **部分止盈**: 盈利7%，平仓50%
3. **全部止盈**: 盈利16%，全部平仓
4. **移动止损**: 盈利4.5%后激活，距离2.7%
5. **移动止盈**: 盈利5.5%后激活，回撤2.2%触发

## ⚠️ 风险提示

1. **杠杆风险**: 使用2.7倍杠杆，需严格风控
2. **严格止损**: 必须执行3.2%硬止损，不可抱有侥幸心理
3. **心态管理**: 遇到连续亏损立即停止，复盘后再交易
