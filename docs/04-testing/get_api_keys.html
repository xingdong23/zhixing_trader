⚙️ 配置步骤

  1. 获取 Finnhub API Key

    访问 https://finnhub.io/register

    • 支持 Google 账号登录

    • 验证邮箱后即可获取

  2. 获取 Twelve Data API Key

    访问 https://twelvedata.com/pricing

    • 选择 "Free Plan"

    • 无需信用卡

  3. 配置环境变量

    编辑 .env 文件：

    # 数据源模式 MARKET_DATA_PROVIDER=multi # API Keys FINNHUB_API_KEY=your_finnhub_key_here TWELVEDATA_API_KEY=your_twelvedata_key_here ALPHA_VANTAGE_API_KEY=AU1SKLJOOD36YINC # 数据源配置 (数据源名:优先级:权重) DATA_SOURCES_CONFIG=finnhub:1:40,twelvedata:1:30,alphavantage:2:15,yahoo:3:15
  4. 测试验证
    cd /Users/chengzheng/workspace/chuangxin/zhixing_trader PYTHONPATH=./zhixing_backend python zhixing_backend/scripts/test_multi_providers.py
  5. 重启服务
    cd zhixing_backend python -m uvicorn app.main:app --reload --port 8000
